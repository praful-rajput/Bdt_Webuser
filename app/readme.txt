This extension is useful for multi-website portal.

It help to display only applicable website's  orders for those adimin users.

Note: Website name and role name should be same. So All user under those role can see only it's website orders.

For Obtain the website names. 
	Go to System >> Manage Stores

Step 1: Add a User Role (Role Name & website name must me same. (equal))
		1. Add user under newly created Role.
	
Step 2: Add New User
		Go to System > Permissions > Users.
		Click the Add New User button.
		In the User Info area, fill in all fields for Account Information.
		Set the account status to Active to enable access to the user.
		Click the User Role tab on the left panel and assign this user to a predefined role.
		Click Save User.
